
# CovidTracker

* Based on MVVM Architecture
* Used Retrofit and Coronva Cases Restful Api

Covid Api : https://corona.lmao.ninja/v2/countries

* Show Details of Corona virus cases of all affected country
      - Total Cases
      - Active Cases
      - Death
      - Recovered
      - Tests
